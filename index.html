Whisper Server Setup

To transcribe audio files, you need to run a local server. Follow these steps once:

1

Install Python

Download from python.org/downloads

During install, check "Add Python to PATH"

2

Install FFmpeg

Open Command Prompt and run:

winget install ffmpeg

Mac: brew install ffmpeg

3

Install Python packages

pip install flask flask-cors openai-whisper

This may take a few minutes

4

Download whisper_server.py

Click here to view → Click "Raw" → Save as whisper_server.py

5

Start the server

py whisper_server.py

Keep this window open while using file upload

Once the server shows "Running on http://localhost:5000", come back here and upload your file!
